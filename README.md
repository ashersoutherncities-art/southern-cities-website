# Southern Cities Enterprises

Official website for Southern Cities Enterprises - a vertically integrated real estate and financial ecosystem.

## Divisions

- **General Contracting** - Professional subcontractor management and project coordination
- **Acquisitions & Deal Sourcing** - Strategic property acquisition and deal analysis
- **Brokerage** - Licensed real estate services
- **Development** - Ground-up construction and rehabilitation

## Tech Stack

- Static HTML/CSS/JavaScript
- Tailwind CSS (via CDN)
- Google Fonts (Inter)
- Fully responsive design
- Optimized for performance

## Deployment

Deployed via GitHub Pages at: https://ashersoutherncities-art.github.io/southern-cities-website/

## Local Development

Simply open `index.html` in a browser - no build process required.
